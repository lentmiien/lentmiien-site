Currently I only support the values "image_generation" and "web_search_preview", or none or both (specified in the `metadata.tools` array in the conversation object). Just append a short generic text like "You can use {tool} if needed" to the context prompt.

Important: Non of these context prompt injections should persist, and the injection process should be repeated for every query for an AI response.
