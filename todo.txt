I've added a new OCR API on my home network, at the URL `192.168.0.20:8000`, with the 2 endpoints `/health` and `/ocr`.

GET `/health` returns a JSON object as following:
```
{
  "status": "ok",
  "model": "tencent/HunyuanOCR"
}
```

POST `/ocr` takes the following input in the body:
```
{
  "file": string($binary),
  "prompt": string | (string | null),
  "max_new_tokens": integer
}
```

"Detect and recognize text in the image, and output the text coordinates in a formatted manner." is a good default value for "prompt", and 2048 for "max_new_tokens".
