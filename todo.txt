I want to create a simple YAML viewer in my app, by adding a new route/controller.

I want to use the `swagger-ui-express` npm library, and the functionality should be:
* A landing page, that shows a list of all files in the `public/yaml` folder, when the user clicks a file, they'll be taken to the viewer page for that file
* A viewer page, to view the selected documentation

Can this be done?

I have the following example code:
```
const express = require('express')
const fs = require('fs')
const YAML = require('yaml')
const swaggerUi = require('swagger-ui-express')
const app = express()
const specText = fs.readFileSync('./openapi.yaml', 'utf8')
const spec = YAML.parse(specText)
app.use('/docs', swaggerUi.serve, swaggerUi.setup(spec, { explorer: true, swaggerOptions: { validatorUrl: null } }))
app.listen(3000, () => console.log('Docs: http://localhost:3000/docs'))
```

Or if using URL:
```
const express = require('express')
const fs = require('fs')
const YAML = require('yaml')
const swaggerUi = require('swagger-ui-express')
const app = express()
const specText = fs.readFileSync('./openapi.yaml', 'utf8')
const spec = YAML.parse(specText)
app.get('/openapi.json', (req, res) => res.json(spec))
app.use('/docs', swaggerUi.serveFiles(null, { swaggerOptions: { url: '/openapi.json' } }), swaggerUi.setup(null, { explorer: true }))
app.listen(3000, () => console.log('Docs: http://localhost:3000/docs'))
```
